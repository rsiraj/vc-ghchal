1. Explain what the terminal is in your own words.
Terminal is a command-line system that helps us quickly take control of the Operating System and make changes without a GUI

2. What is the major difference between Git and Github?
Git is a local version control system that can be on any computer whereas github is a place for online repos where all the gits can be stored online to make them available to anyone

3. Why do we use Github?
We use it because it's free and open source. It helps us collaborate and do version control.
